# v0.1.0

Initial public draft release of the Multi-Level Repository Navigation and Routing Convention.

## Highlights

- defines a multi-level repository entrypoint model built around `AGENTS.md`, `ENTERPRISE.md`, `SOLUTION.md`, and `DOMAIN.md`
- defines deterministic routing catalogs for enterprise-to-solution, solution-to-domain, and domain-to-implementation handoff
- separates lightweight navigation entrypoints from canonical machine-readable routing artifacts
- includes `Core` and `Governed` conformance profiles
- includes starter templates and complete example layouts
- includes Claude Code bridging guidance via `CLAUDE.md` templates while keeping `AGENTS.md` as the cross-tool convention

## Included In This Release

- canonical spec: `enterprise_repo_convention.md`
- repository overview: `README.md`
- starter templates and `ea-convention` skill under `skills/ea-convention/`
- example repositories under `examples/core/` and `examples/governed/`
- project governance and contribution files for open-source publication

## Release Notes

- status remains `Draft`
- project version is `0.1.0`
- this release is intended as an initial public baseline for feedback and iteration, not a stable final standard
